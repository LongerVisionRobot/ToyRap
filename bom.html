* - Some parts can be optionaly rendered as glue-on or screw-on versions. I used glue-on but if you prefer screw-on you will need additional bolts and hardware to attach them.

** - Some parts can be redered with optional supports. This can be useful with slicers that do not allow limiting supports to specific areas. Many printers will probably do fine without any supports so try w/o first.

*** - The text "ToyRap" on the lcd box is editable within the OpenSCAD source file. If you are familiar with OpenSCAD you may wish to edit and render this one from scratch even if using pre-rendered 3MF or STL files for everything else.
